# Okovision Android

Application Android connectée à Home Assistant pour le suivi en direct d’une chaudière Okofen, de la consommation de pellets et du stock sur palette.

## Fonctionnalités
- Lecture chaudière (HTTP local / JSON / CSV)
- Stock de sacs en MQTT Discovery
- Synchronisation Home Assistant via MQTT
- Graphiques MPAndroidChart (conso, température)
- Stockage local via Room + DataStore

## Version
v2.3.5 – Kotlin 2.0, Compose Material3, Java 17
